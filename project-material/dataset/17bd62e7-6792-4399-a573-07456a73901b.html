For hyperparameter tuning of the Decision Tree Classifier, two commonly tuned hyperparameters are:

  1. max_depth:

    • Definition: The maximum depth of the tree. It controls the maximum number of levels of nodes from the root node. A higher value can lead to a more complex model.
    • Reason for tuning: Adjusting the max_depth can help prevent overfitting. Too deep a tree may capture noise in the training data, so tuning this hyperparameter is crucial to finding the right balance between complexity and generalization.

  2. min_samples_split:

    • Definition: The minimum number of samples required to split an internal node. It controls the minimum number of samples a node must have to be split. Smaller values create more complex trees.
    • Reason for tuning: Adjusting min_samples_split helps prevent the model from being too sensitive to the noise in the data. It can contribute to making the model more robust and less prone to overfitting.

Here's how you can use GridSearchCV to tune these hyperparameters:

python
from sklearn.tree import DecisionTreeClassifier
from sklearn.model_selection import GridSearchCV

8) Find the information gain on the first split (10 pts) Find the information gain on the first split with Entropy according to the formula from the lecture notes (given below) information gain = entropy(parent)-averageentropy(children)
ChatGPT

To find the information gain on the first split with entropy, you need to calculate the entropy of the parent node and the average entropy of the child nodes. Here's how you can do it:

python
from sklearn.tree import _tree

# Get the tree structure
tree_structure = best_dt_model.tree_

# Define a recursive function to calculate entropy
def calculate_entropy(node):
    value = tree_structure.value[node].flatten()
    value = value[value != 0]  # Remove zero counts

    prob = value / value.sum()
    entropy = -np.sum(prob * np.log2(prob))
    return entropy

# Calculate entropy for the parent node
entropy_parent = calculate_entropy(0)  # 0 corresponds to the root node

# Calculate entropy for each child node
entropy_children = []
for child in (tree_structure.children_left, tree_structure.children_right):
    entropy = np.sum([calculate_entropy(node) for node in child if node != _tree.TREE_LEAF])
    count = len([node for node in child if node != _tree.TREE_LEAF])
    entropy_children.append((entropy, count))

# Calculate the average entropy of the children
average_entropy_children = np.average([entropy * (count / sum([count for entropy, count in entropy_children])) for entropy, count in entropy_children])

# Calculate information gain
information_gain = entropy_parent - average_entropy_children

print("Information Gain on the first split with Entropy:", information_gain)

This code extracts the tree structure from the trained Decision Tree model, calculates the entropy for the parent node (root), and then computes the entropy for each child node. Finally, it calculates the information gain based on the provided formula.
